Description: 用户表

1.4.1. Columns

Column name Type Properties Description
id bigint PK
name varchar(15) null 用户名

1.5. Table question

Description: 问题表

1.5.1. Columns

Column name Type Properties Description
id bigint PK
uid bigint null 出题人的id
type int null 题型 选择1 判断2 填空3
q varchar(255) null 问题内容
A char(1) 选项A
B char(1) 选项B
C char(1) 选项C
D char(1) 选项D
F char(1) 正确/错误选项 只可能有T/F两种值
correct varchar(10) null 正确答案
majorID bigint null 所在分类ID
challenges int null 挑战人数
passed int null 通过人数
levels int null 问题难度星级
balels varchar(255) 标签 多个用逗号分开 最多5个
toAnswer varchar(50) null 给答题者的话，最多50字
